# rn-schedule-clear-cache

Clear cache regularly using AlarmManager

## Installation

```sh
npm install rn-schedule-clear-cache
```

## Usage


```js
import { multiply } from 'rn-schedule-clear-cache';

// ...

const result = await multiply(3, 7);
```
